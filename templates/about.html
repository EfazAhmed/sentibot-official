PURPOSE

By using a dataset of various user inputs categorized into emotion groups, we trained a computer model to correctly identify a user’s emotions based on their input. SentiBot will utilize the data model to output a relevant response for the user’s emotions, along with an emotion response reading in percentages.

DATA

By using multiple Kaggle datasets here and here, we've trained a Random Forest Classification Model to create a bot that uses sentiment analysis to output and display the range of emotions conveyed through user's input. The data used to create the model was labeled correctly, but needed to be cleaned in order to attain optimal results.
